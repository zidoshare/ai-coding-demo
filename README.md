# Vibe Coding Demo

基于 GLM 4.7

## 如何运行

去 <https://bigmodel.cn/usercenter/proj-mgmt/apikeys> 获取 GLM API Key

```
git clone https://github.com/zidoshare/vibe-coding-demo.git
cd vibe-coding-demo
npm install
npm run dev
```

访问 `localhost:3000`，默认密码是 test@test.com/123456
